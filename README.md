# 🎓 紫熊祭 入ステ班 引継ぎポータル (Hikitsugi Portal)

熊本大学 学園祭（紫熊祭）実行委員会「入ステ班」のための、業務引継ぎ・進捗管理ポータルサイトです。
**実働約12時間**で要件定義から実装、デプロイまでを完遂したプロトタイプです。

## 🚀 Live Demo
[https://nyuusute-hikitsugi.netlify.app/](https://nyuusute-hikitsugi.netlify.app/)

## 🛠 開発の動機 (Motivation)
これまでの引継ぎ業務には、以下の**「情報の非対称性と検索コスト」**という課題がありました。

1.  **情報の散逸**: 必要な情報が「膨大なPDF資料」や「去年のLINEグループのログ」に埋もれており、探すだけで時間がかかる。
2.  **属人化**: 「これどうやるんでしたっけ？」と毎回先輩や部長に確認する必要があり、双方にとってタイムロスが発生していた。
3.  **現状把握の困難さ**: 過去のデータと比較して、現在の進捗が順調なのか遅れているのかを客観的に判断する術がなかった。

これらの「めんどくさい」作業を全て自動化・集約し、**「これを見れば全てわかる」状態を作るため**に開発しました。

## ✨ 主な機能 (Features)

### 📊 1. 進捗管理・比較シミュレーター (Progress Simulator)
- **ロジック**: 制作物ごとに「重み（工数）」を設定。昨年度（第14回）の同日時点のデータと、現在の進捗値をリアルタイムで比較します。
- **自動判定**: 「昨年より遅れている」「順調」などのステータスを自動判定し、アラートを表示して行動を促します。

### 📁 2. 制作物データベース (Production DB)
- 入場門やステージ看板など、制作物の「正確な寸法（cm）」と「必要資材」をデータベース化。
- 散らばっていた購入先情報や過去の反省点を一元管理。

### 📱 3. モバイルファーストUI
- 買い出し中や作業現場（屋外）でスマホから即座に確認できるよう、徹底したレスポンシブデザインを採用。

## 💻 技術スタック (Tech Stack)

最短時間で「動くもの」をデリバリーすることを最優先し、軽量かつ高速な構成を採用しました。

- **Frontend**: HTML5, Vanilla JavaScript (ES6+)
- **Styling**: Tailwind CSS (CDN)
- **Icons**: FontAwesome
- **Deploy**: Netlify
- **AI Collaboration**: ChatGPT / Claude (要件定義・ロジック構築・実装の高速化に使用)
